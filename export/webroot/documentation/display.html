display([message])

The display-command displays a text/images/tables etc. on the subjects' displays. This command supports full HTML-syntax for formatting the output if required.

Parameters

messageOptional. The message to be displayed.
If no message is supplied, an empty line will be inserted on the subjects' displays.
The message can contain most HTML-Code (see below for examples) like text formatting, tables etc.. Frames and applets will not work, however.
A good documentation of HTML including many examples can be found here.
If you want to use the characters "<" and ">", please use their HTML equivalents "&lt;" and "&gt;".

Including images

Note that while text is scaled according to the screen resolution, images are not.
If you use images from internet sources, make sure that each computer used in the experiment has access to the internet.
If you use local images, make sure that every computer used in the experiment can access the image at the given path. You can either use a network drive or, alternatively, copy the images to every used computer.
