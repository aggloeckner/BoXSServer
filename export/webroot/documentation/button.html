button(variablename,message)

The button-command creates a button on each subject's display. If the subject clicks on the button, the specified variable will be set to 1. These buttons are much more flexible and versatile than the buttons created by the wait-commands. One possible application are loops which are to be left when a subject clicks on a button.

Parameters

variablename The name of the variable which is set to 1 when the button is pressed.
message The message to be displayed on the button.
