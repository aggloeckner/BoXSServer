recordKeys()

The recordKeys-command can be used to directly react to a user pressing a key. This can be useful, for example, for experiments with people being examined by an fMRI, who can only interact with the experiment computer using a few buttons. When putting the recordKeys-command in a stage, the key code of the key which was pressed is stored in the variable _lastkey which can then be used.
In a stage using recordKeys, no other input elements should be used as they could interfere with the keyboard focus which is required for recordKeys to work.
The exact behaviour of keyboard focus for applets is not defined and can depend on Java version, internet browser etc.. In that sense, recordKeys should be avoided in internet experiments in which the subjects' computer configuration and concurrent processes are not known.
