locale(localcode)

You can set the localisation settings for your experiment (e.g. the number formatting pattern) by using the locale command. The locale also effects the default text of the wait/continue-buttons, however, only German and English are implemented for these by default.
The locale codes can be looked up here. Examples include "de" for German, "en" for English and so on.
The default which is used when no locale command is included in your program is English.
For more information on localising text in multilingual experiments have a look at this example program.
