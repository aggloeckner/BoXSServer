clear()

The clear-command can be used to clear user input when using for- or while-loops.By default the input elements like text boxes are not reset when the next iteration of a loop is shown. Therefore a subject's previous input is still visible when the next round is shown. This may not be desired as it might influence the subject's decisions. In order to avoid this put the clear-command at the beginning of the loop.
The reason for the unintuitive default behaviour is that loops are also used for high frequency experiments like auctions and it would be very disturbing if all user input would be cleared with every refresh.
