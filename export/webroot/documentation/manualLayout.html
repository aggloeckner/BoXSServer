manualLayout() and automaticLayout()

Usually the automatic layout of the components is both convenient and sufficient. In some cases, however, more control over the component layout is required. In these cases you can use the manualLayout()-command. This command enables you to specify each component's position and size on a pixel basis.
You only need to specify manualLayout once at the beginning of the program. If you want to restore the automatic layout, use the automaticLayout()-command. Both should be done at the beginning of a screen, i.e. directly at the beginning of a program of after a wait-command.
The position and size are appended to the parameters of the component in question. For example, for the display-command the syntax is now display(text, position x, position y, width, height). Once you specify manualLayout you must do this, otherwise the respective components will not be displayed.
