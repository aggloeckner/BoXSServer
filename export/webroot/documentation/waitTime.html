waitTime(time)

The waitTime-command halts the execution of the program for the specified time.
The server will update its clients twice a second at most for performance reasons. Therefore using waitTime with a time smaller than 500 msecs can lead to unexpected results.
If a refresh frequency higher than 2Hz is required, you have to set up an offline server. For this server you can specify the exact refresh rate as a command line parameter.

Parameters

time The time the program will wait in milliseconds (!).
